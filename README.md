# ember-table-demo

A step-by-step demonstration of creating a simple ember-table and
progressively adding more sophisticated features.

## Installation

- Clone this repository
- `yarn install`
- `ember serve`
- Visit http://localhost:4200 or http://0.0.0.0:4200
